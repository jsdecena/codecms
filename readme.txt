How To install

1. Create a database - 'codecms' . You can change this in the system/config/database.php
2. Dump the sql in your newly created database. The SQL dump is found in codecms/assets/sql/db.sql
3. Change the setting of your database settings in system/codecms/config/database.php
4. Login url for the admin is http://example.com/admin
4. Default user is Admin. Login details are as follows - Email: admin@admin.com Password: 123123
